# 监控安全容器

## 描述

kata events命令用于显示指定容器状态。包括但不限于容器内存、CPU、Pid、 Blkio、大页内存、网络等信息。

## 用法

```
kata-runtime metrics <sandbox id>
```

## 前置条件

要查询的容器状态必须为running，否则报错：Container ID \(<container\_id\>\) does not exist。

该命令只支持查询监控一个容器的状态
